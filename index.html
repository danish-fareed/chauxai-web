Recommended: Download the standalone ChauxAI Agent Binary (.exe) (27.2 MB) or the Desktop Setup Installer.
Or run manually from source:
1
Install Python dependencies
In your automation_scripts folder, run:
pip install -r requirements.txt
2
Start the local agent
Run: python app.py — it will start on port 5000
3
Enter your agent URL below
Use http://localhost:5000 if this browser is on the same PC, or your local network IP like http://192.168.1.x:5000
This URL is saved in your browser and never sent anywhere. You can change it anytime via Agent Connection in the sidebar.
